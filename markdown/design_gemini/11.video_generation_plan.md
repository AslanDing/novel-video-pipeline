## 流程验证

### 数据流
```
Stage 1: script_x.jsonl
    ↓
Stage 2: 图像生成 → images/
    ↓
Stage 3: 音频生成
    → TTSScriptAdapter 读取 script_x.jsonl
    → 生成 audio/voices/*.wav
    → TimelineGenerator 生成 timeline_chXX.json
    ↓
Stage 4: 视频合成
    → VideoComposer._load_timeline() 读取 timeline_chXX.json
    → 使用时间线精确对齐图像和音频
    → 生成最终视频
```

### 完整流程已打通！
```
Stage 1 (小说) → script_x.jsonl
    ↓
Stage 2 (图像) → 使用 ScriptAdapter 读取 script_x.jsonl → images/
    ↓
Stage 3 (音频) → 使用 TTSScriptAdapter 读取 script_x.jsonl
               → 使用 TimelineGenerator 生成 timeline_chXX.json → audio/
    ↓
Stage 4 (视频) → 读取 timeline_chXX.json → 精确合成视频
```
