## 下一阶段：蓝图生成拆分 (Phase 2) ⚠️

### 当前问题
当前 `_create_blueprint` 一次性生成完整蓝图，Prompt 内容过多，导致：
1. 生成质量不稳定
2. 某些字段被压缩省略
3. 难以精细控制各部分

### 拆分方案
```
Step 1: 生成世界观 (WorldBuildingOutput)
   ↓
Step 2: 生成角色 (CharactersOutput) - 基于世界观
   ↓
Step 3: 生成修炼体系 (PowerSystemOutput) - 基于世界观
   ↓
Step 4: 生成情节结构 (PlotStructureOutput) - 基于角色和世界观
   ↓
Step 5: 生成章节规划 (ChapterPlansOutput) - 批量生成
```

### 新增文件
- `stages/stage1_novel/pydantic_models.py` - 添加拆分用的 Output 模型
- `stages/stage1_novel/prompts/protocol_prompts.py` - 添加拆分用的提示词模板

- ✅ `StoryBlueprint` - 包含 world_building, characters, plot_structure, chapter_plans

```python
# stage1_novel/models.py
class ScriptLine(BaseModel):
    scene_id: str           # 场景ID（如 SC01）
    shot_id: str            # 镜头ID（如 SC01_SH01）
    role: str               # dialogue 或 narrator
    speaker: str            # 发声者名称
    text: str               # 朗读文本
    emotion: str            # 情绪标签
    visual_prompt: str      # 图片生成英文提示词
    motion_prompt: str      # 镜头运动指引
    camera: str             # 景别描述
    estimated_duration: float  # 预估时长(秒)
```
