# Stage 4: 视频合成引擎 - 简化本地版

## 概述

将Stage 2生成的图像和Stage 3生成的音频合成为最终视频，支持字幕叠加和简单的转场效果。

## 技术选型配置

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| **视频生成模型** | HunyuanVideo-1.5 I2V | 腾讯开源图生视频 |
| **分辨率** | 480p (854x480) | 24GB单卡友好 |
| **帧率** | 24 fps | 电影标准 |
| **生成时长** | 3-5秒/镜头 | 短镜头保证稳定性 |
| **运动强度** | 中等 (motion_bucket=127) | 平衡动感和稳定 |

### 备选方案 - Ken Burns效果

当显存不足或I2V失败时，使用FFmpeg实现静态图动态效果：

```bash
# 缓慢推镜头 (Slow push-in)
ffmpeg -i image.png -vf "zoompan=z='min(zoom+0.0015,1.5)':d=125:x='iw/2-(iw/zoom/2)':y='ih/2-(ih/zoom/2)'" output.mp4
```

**视频合成参数**:

| 参数 | 值 | 说明 |
|------|-----|------|
| 编码器 | H.264 (libx264) | 兼容性最佳 |
| 预设 | medium | 速度/质量平衡 |
| CRF | 23 | 视觉无损 |
| 像素格式 | yuv420p | 兼容性 |

## 输入输出

### 输入
- 场景图像: `outputs/images/{chapter}/{scene}.png`
- 分段音频: `outputs/audio/{chapter}/{segment}.mp3`
- 小说JSON: `outputs/novels/{name}/novel.json`
- 字幕文件(可选): `outputs/audio/{chapter}/{segment}.srt`

### 输出
- 合成视频: `outputs/videos/{chapter}/chapter_{n}.mp4`
- 带字幕视频: `outputs/videos/{chapter}/chapter_{n}_subtitled.mp4`
- 视频元数据: `outputs/videos/{chapter}/metadata.json`

## 核心组件

### 1. VideoComposer (视频合成器)

```python
class VideoComposer:
    """视频合成主类"""

    def __init__(self, config: VideoGenConfig):
        self.config = config
        self.ffmpeg_path = config.ffmpeg_path
        self.resolution = self._parse_resolution(config.resolution)
        self.subtitle_generator = SubtitleGenerator()
        self.transition_engine = TransitionEngine()

    def compose(
        self,
        chapter_num: int,
        scenes: List[VideoScene],
        output_path: Path,
        add_subtitles: bool = True
    ) -> VideoOutput:
        """
        合成视频

        Args:
            chapter_num: 章节号
            scenes: 视频场景列表
            output_path: 输出路径
            add_subtitles: 是否添加字幕

        Returns:
            VideoOutput: 合成结果
        """
        # 检查缓存
        cache_key = self._get_cache_key(chapter_num, scenes)
        cached = self.cache_manager.get(cache_key)
        if cached and not self.config.force_regenerate:
            return VideoOutput(**cached)

        # 准备输入文件
        input_file = self._create_ffmpeg_input(scenes)

        # 构建FFmpeg命令
        if add_subtitles:
            video_path = self._compose_with_subtitles(
                input_file=input_file,
                scenes=scenes,
                output_path=output_path
            )
        else:
            video_path = self._compose_without_subtitles(
                input_file=input_file,
                scenes=scenes,
                output_path=output_path
            )

        # 验证输出
        if not video_path.exists():
            raise RuntimeError("视频合成失败")

        # 获取视频信息
        video_info = self._get_video_info(video_path)

        # 构建结果
        result = VideoOutput(
            video_path=str(video_path),
            chapter_number=chapter_num,
            total_scenes=len(scenes),
            duration_seconds=video_info['duration'],
            resolution=video_info['resolution'],
            file_size_mb=video_path.stat().st_size / (1024 * 1024)
        )

        # 保存缓存
        self.cache_manager.set(cache_key, result.dict())

        return result

    def _generate_srt(self, scenes: List[VideoScene], output_path: Path) -> None:
        """生成SRT字幕文件"""
        srt_entries = []
        current_time = 0.0

        for i, scene in enumerate(scenes):
            if not scene.subtitle_text:
                current_time += scene.duration_seconds
                continue

            # 计算时间码
            start_time = current_time
            end_time = current_time + scene.duration_seconds

            # 格式化时间码
            start_srt = self._format_time(start_time)
            end_srt = self._format_time(end_time)

            # 构建SRT条目
            srt_entries.append(f"{i+1}")
            srt_entries.append(f"{start_srt} --> {end_srt}")
            srt_entries.append(scene.subtitle_text)
            srt_entries.append("")

            current_time = end_time

        # 写入文件
        output_path.write_text('\n'.join(srt_entries), encoding='utf-8')

    def _format_time(self, seconds: float) -> str:
        """格式化为SRT时间码"""
        hours = int(seconds // 3600)
        minutes = int((seconds % 3600) // 60)
        secs = int(seconds % 60)
        millis = int((seconds % 1) * 1000)
        return f"{hours:02d}:{minutes:02d}:{secs:02d},{millis:03d}"
```

## 命令行接口

```bash
# 完整视频合成
python run_stage4.py --novel-dir "outputs/novels/我的修仙小说"

# 只合成指定章节
python run_stage4.py --novel-dir "outputs/novels/我的修仙小说" --chapter 1

# 不使用字幕
python run_stage4.py --novel-dir "outputs/novels/我的修仙小说" --no-subtitles

# 强制重新生成
python run_stage4.py --novel-dir "outputs/novels/我的修仙小说" --force

# 指定分辨率
python run_stage4.py --novel-dir "outputs/novels/我的修仙小说" --resolution 1920x1080

# 指定FPS
python run_stage4.py --novel-dir "outputs/novels/我的修仙小说" --fps 30
```
