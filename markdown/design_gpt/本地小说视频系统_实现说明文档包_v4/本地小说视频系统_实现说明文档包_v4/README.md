## 文档定位

这一包文档在 v3 详细设计文档的基础上继续下钻，目标不是重新讲概念，而是补足：

- 端到端时序和泳道
- 每个阶段的输入输出字段规范
- 任务状态转移、重试、幂等、补偿
- GPU 调度算法与资源控制
- ComfyUI、FFmpeg、Prompt Rewriter 的落地 workflow
- 数据库存储、迁移、审计和回滚
- 部署拓扑、启动顺序和运行手册
- 质量评分、人工审核与发布 gate

本包文档建议与 v3 文档包联合阅读：
- 先看 v3 的架构、领域模型、子系统说明
- 再看 v4 的实现说明与执行细节

## 这一版的输出目标

看完这一包，开发团队应该能够：

1. 明确一次完整章节生产的执行顺序
2. 明确每个节点的输入输出和状态转移规则
3. 明确何时缓存、何时重试、何时人工介入
4. 明确单 GPU 场景下的模型加载和排队策略
5. 明确如何落地数据库、文件系统和产物索引
6. 明确如何启动环境并进行联调
7. 明确质量门禁和发布标准
